## Запуск 
TTL запросов и максимальное количество переходов по ссылке можно задать в файле
[application.yml](src/main/resources/application.yml)

Запуск приложения
```
./gradlew bootRun
```

Запуск тестов
```
./gradlew test
```

## Тестирование
По url http://localhost:8080/swagger-ui/index.html доступен сваггер, там можно сгенерировать короткую ссылку.
Выполнить редирект сваггер не даст, проверить корректность перехода можно с помощью curl:

```
curl -v -H "userId: <Тут uuid>" <Тут короткая ссылка>
```

Пример:
```
curl -v -H "userId: 3fa85f64-5717-4562-b3fc-2c963f66afa6" http://localhost:8080/IlqsF8a
```

Лимит переходов по ссылке и время жизни задаются в application.yml.
Сообщения об истечении лимита переходов или времени жизни пишутся в лог, при подобных сценариях сервер вернет 404.

## Проверяющим
Операции редактирования/удаления ссылок в тз не описаны, но почему-то есть в критериях, их не реализовывал, обратную связь команде курса передам.
Консольное приложение не может быть многопользовательским! 
Критерии проверки противоречат друг другу: есть требование к взаимодействию одновременно с несколькими пользователями и есть требования к CLI, прошу не снижать быллы за cli, т.к. выполнить сразу все требования теоретически невозможно.
